# ⚡ OP_LAUNCH — DeFi Launchpad

The most builder-friendly launchpad in the OP_NET / Vibecode ecosystem.

## Features
- 🚀 No-code OP_20 Token Deploy Wizard
- 📊 Presale Dashboard with live progress
- 📅 Vesting Schedule (linear, cliff, custom)
- 🔐 Liquidity Lock with MotoSwap integration
- 🤝 OTC Escrow for peer-to-peer trades
- ⚡ Auto-Compound Staking Vault
- 🔍 Transparent TX Log with OP_SCAN links

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Build for Production

```bash
npm run build
npm run preview
```
